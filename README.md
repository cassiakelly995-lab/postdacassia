## 📌 O que é este projeto?

Uma **biblioteca interativa de conteúdo** para TikTok e Instagram no nicho de reflexão e desenvolvimento pessoal.

Cada entrada da biblioteca contém:
- 📝 **Caption pronto** — texto formatado para feed ou script para TikTok
- 🎨 **Prompt Leonardo AI** — em português e inglês, calibrado para imagens sem pessoas em formato 9:16
- 🏷️ **Hashtags** — já organizadas por categoria
- 📋 **Botão de cópia** — copiar post ou prompt direto para área de transferência

Tudo pronto para **copy-paste direto no feed**.

## 🎯 Formatos por post

Cada post está classificado por formato de uso:

- **Post / Caption** → texto para feed Instagram ou legenda TikTok
- **Script TikTok** → roteiro narrado (15–30s)
- **Carrossel** → estrutura slide a slide com CTA final
- **Script POV** → roteiro em primeira pessoa

---

## 🖼️ Prompts Leonardo AI

Todos os prompts foram desenvolvidos com os seguintes critérios:

- **Sem pessoas** — foco em objetos, natureza e cenários
- **Formato 9:16** — otimizado para Reels e TikTok
- **Modelos indicados** — Leonardo Diffusion XL ou Leonardo Kino XL
- **Parâmetros sugeridos** — Steps (40–50) e Guidance (7–8) para cada prompt
- **Versão PT + EN** — descrição em português e prompt técnico em inglês
